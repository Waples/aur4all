##### Commands Cheatsheet
* [core]
- -S    Sync internal mirrorlist
- -Q    Query installed PKGs
- -R    Remove PKG(s)
- -U    Update installed PKGs

* [args]
- -s    Search.
- -i    Install.
- -y    Same as *-i* but without confirmation.


##### Examples
```
aur4all -Ss *package*
  > Searches the AUR for query
```
